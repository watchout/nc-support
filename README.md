## 5. Codex デプロイ手順
1. Codex ▶ **New Project → Git Import**（このリポジトリ）  
2. Build Command = `none`, Output Dir = `./`  
3. 必要なら Assets に画像をアップし `src="/assets/…"` へ修正  
4. 環境変数例 LINE_ID=https://lin.ee/XXXX
5. **Publish** で反映
